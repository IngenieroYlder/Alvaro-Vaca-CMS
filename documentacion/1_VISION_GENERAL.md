# Visión General del Proyecto - Patios CMS

## Introducción
Este documento detalla el estado actual y las funcionalidades del sistema "Patios", un CMS (Sistema de Gestión de Contenidos) personalizado y modular desarrollado para **Colombia Pictures**. El sistema permite la administración de usuarios, roles dinámicos, permisos granulares y una personalización visual profunda (Temas y SEO).

## Tecnologías Principales

### Backend (Servidor)
- **Framework**: [NestJS](https://nestjs.com/) (Node.js)
- **Lenguaje**: TypeScript
- **Base de Datos**: PostgreSQL
- **ORM**: TypeORM
- **Autenticación**: JWT (JSON Web Tokens)

### Frontend (Cliente)
- **Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Lenguaje**: TypeScript
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Iconos**: [Lucide React](https://lucide.dev/)
- **Animaciones**: Framer Motion
- **Cliente HTTP**: Axios

## Estado Actual
El sistema cuenta con un núcleo funcional robusto que incluye:
1.  **Seguridad**: Login seguro y protección de rutas.
2.  **Administración**: Gestión completa de usuarios y roles.
3.  **Personalización**: Un motor de temas que controla la apariencia y el SEO del sitio en tiempo real.
4.  **UI/UX**: Interfaz moderna, responsiva y con retroalimentación visual (modales, alertas).
5.  **Motor de Candidatos**: Portal de empleo completo con funcionalidades de autogestión para postulantes.
